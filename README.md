# Raycaster Project (Raylib)

This repository contains a university course project focusing on a raycasting-based application and its documentation.

---

## Project Website

The project website is hosted via GitHub Pages and is accessible at:

https://sorina84.github.io/raycaster-raylib/

The website provides:
- An overview of the project
- A development log documenting implementation stages
- A gallery of visual outputs
- A structured and styled static interface

---

## Version Control and Branching

- main  
  Contains the final and stable version of the project, including the GitHub Pages source located in the /docs directory.

- web-design  
  Used for website-related development and merged into the main branch through a pull request.

---

## Technologies Used

- HTML5  
- CSS3  
- Git  
- GitHub  
- GitHub Pages  

---

## Additional Notes

- The /docs directory is configured as the publishing source for GitHub Pages.
- The project website is fully static and does not rely on external frameworks.
- All required deliverables for the course submission are included in this repository.
